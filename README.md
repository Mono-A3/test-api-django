# 🛠️ API - Django REST

API desarrollada con **Django REST Framework** para manejar los endpoints del proyecto.  
Se utiliza **Gunicorn** y **WhiteNoise** para la ejecución y los archivos estáticos en producción.

## 🚀 Tecnologías

- Python 3.11
- Django 5.2.4
- Django REST Framework
- Gunicorn
- WhiteNoise
